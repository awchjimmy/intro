# Bug Bounty Hunter

### day 1 - 設定目標
1. bugcrowd 平台找到 2-3 個 web 目標
2. 一個月內 submit 第一個 report
3. 二個月內有 3-5 個 report，至少一個 triage 有 bounty
4. 二個月內 $150，第三個月開始 $100/月
5. 初期平日每天開，中期一週至少二次直播

> 希望以後成為一個平台、一個社群
