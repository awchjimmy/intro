# 詩歌

## 破冰
### 一切歌頌讚美 - 快
```
一切歌頌讚美 都歸我主我的神
你是配得歌頌與讚美
我們高聲呼喊 高舉耶穌聖名
哈利路亞

(副歌)
讚美主 哈利路亞
讚美主 哈利路亞
哈利路亞
```

## 信心、天堂、希伯來書
### 這世界非我家 - 快
```
這世界非我家 我無一定住處
我積財寶在天 時刻仰望我主
天門為我打開 天使呼召迎迓
故我不再貪愛 這世界為我家

(副歌)
我主 你知 我無良友像你
天堂若非我家 我必流離無依
天門為我打開 天使呼召迎迓
故我不再貪愛 這世界為我家
```

## 親近耶穌、寶貴耶穌
### 耶穌你是寶貴 - 慢
```
耶穌你是寶貴 你對我真寶貴
你流寶血洗淨我 你使我自由

(副歌)
我只想天天讚美你
我只想要彰顯你的名
主阿我愛你 我要更愛你

我只想一生來事奉你
我只想要榮耀你的名
主阿我愛你 我要更愛你
```
